### Description

给你一个有k条边的二分图，每条边端点为n,m，问你存在多少个四元组（A，B，C，D）使得A->B->C->D可行（A,B,C,D为二分图上的点，即有A->B->C->D这样一条有向路）。

数据范围：

多组数据，T <= 1000

1 <= n,m,k <= 1e5

### Sample

##### Input

```
3
2 2 4
1 1
1 2
2 1
2 2
3 1 3
1 1
2 1
3 1
3 3 3
1 1
2 1
2 2
```

##### Output

```
8
0
2
```

### Solution

路径长度为3，枚举每条边(u,v)，以为这条边为中间边的方案数为2*deg[u-1] * deg[v-1]，segma起来即可。
