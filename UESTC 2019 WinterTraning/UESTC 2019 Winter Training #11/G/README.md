### Description

有R个接受点，接受一定的量Ei，P个输出点，输出点只能输出一定的量Di，和C条运输边（I,J,T）表示J输出点到I输出点需要T时间，运输也只能运输一定的量，每条运输边会花费一定的时间，问最短时间使得所有接受点都满。无法满足要求则输出-1。

数据范围：

1 <= P,R <= 1000

1 <= C <= 20000

1 <= T <= 1e6

### Sample

##### Input

```
3 2 5 // P R C
20 10 10 //Ei
25 30 //Di
1 1 3
2 1 1
2 2 4
3 1 2
3 2 5
```

##### Output

```
5
```
