### Description

给你一个长度为n的序列，每次你可以选择两个数ai和aj，使得ai = aj = gcd(ai, aj)，让你用最多2N次操作让所有整个序列都变为1；如果无法完成则输出-1。

数据范围：

多组数据

1 <= n <= 1e5

1 <= ai <= 1e9

### Sample

##### Input

```
4
2 2 3 4
4
2 2 2 2
```

##### Output

```
Case 1: 3
1 3
1 2
1 4

Case 2: -1
```

### Solution

先求出所有数的gcd，若所有数的gcd不为1，则无法完成，否则我们先令1与2~n做gcd，这样a[1] == 1, 之后再做一遍，就能使得整个序列都是1。
