### Description

给定长度为N的数组，Q次询问，每次给出长度L，让你按照规定跑。 每次从当前位置出发，一直跑不降序列，或者下降序列。跑完后，如果大于等于L，从下一个位置继续开始，否则要凑齐长度L，同时累计凑数的个数。

数据范围：

5 <= N <= 1e5

1 <= Q <= 1e5

### Sample

##### Input

```
5 8 3 6
```

##### Output

```
16
```

### Solution

预处理出从每个点为起点开始走不上升或者不下降的终点，每次取较远的跳，跳完检查是否够L或者不足补齐。对于L= 2 ~ N都预处理好，由于每次至少跳L步，复杂度是N/2 + N/3 + .. N/N 约为 NlogN。
